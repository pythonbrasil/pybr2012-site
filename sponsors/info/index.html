Patrocinando a PythonBrasil

Interessado em contratar desenvolvedores Python talentosos? Patrocine a conferência e aumente a visibilidade da sua empresa.

  • Plano de captação (versão em inglês)
  • Plano de captação (versão em português)

A PythonBrasil é uma conferência sem fins lucrativos organizada por voluntários para promoção da linguagem de programação de código aberto Python. É apoiado pela Associação Python Brasil (APyB) e pela Python Software Foundation (PSF).
